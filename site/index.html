PM Agent Kit Workbook

20 days. One skill a day. Build your AI-powered PM practice.

What is the PM Agent Kit?

The PM Agent Kit is a suite of PM skills encoded in a portable, agentic system built for Claude Code. It automates the PM work you do most: drafting specs, reviewing documents, planning sprints, synthesizing research, building business cases, and more. This workbook is the structured practice companion: twenty minutes a day, Monday through Friday, for four weeks. PM Agent Kit on GitHub

How it works

Step 1
Set up

Clone the PM Agent Kit and copy the Terrain company context files. Day 0 walks you through everything.

Step 2
Meet Terrain

Get oriented to the fictional outdoor marketplace you'll use as your practice environment throughout the challenge.

Step 3
Practice one skill a day

Each day introduces one PM Agent Kit skill, explains why it matters, and gives you two copy-ready prompts to run in Claude Code. Twenty minutes, then done.

Step 4
Reflect

Three questions at the end of each day to connect what you saw to your real work.

A consistent practice environment

All 20 days use Terrain — a fictional two-sided marketplace for bookable outdoor experiences — so prompts build on each other and you see how skills chain together across a real product context. You don't need to memorize Terrain; all the relevant details are embedded in the prompts.

What "interactive" means here

This workbook is interactive in a specific sense: every prompt has a one-click copy button, every day is linked, and each page has machine-readable metadata embedded for agents. It does not track progress, store state, or require login. Your notes live wherever you keep them.
